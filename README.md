# Movie Discovery App

A React app for browsing and discovering movies, with client-side routing between views (home, details, search, etc.) via React Router.

## Features
- Browse/search movies
- Multi-page navigation with React Router
- Component-based architecture for movie listings and details

## Tech Stack
React 19 · React Router 7 · Vite · ESLint

## Getting Started
```bash
git clone https://github.com/uryapraveen/Movie-App.git
cd Movie-App
npm install
npm run dev
```

## Status
Built to practice React Router and component composition for a multi-view app.
